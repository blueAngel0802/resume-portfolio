# Portfolio Templates (Job + Freelance)

This project is a **portfolio-style** website (not a resume dump). It's optimized for:
- **Jobs**: highlights impact + lets recruiters download the full resume (PDF)
- **Freelance**: shows services, process, and case-study style project cards

## Run
```bash
npm install
npm run dev
```

## Edit content
Open `src/data/portfolio.ts` and edit:
- headline / subheadline
- proof metrics
- project templates (case-study style)
- services & process
- experience highlights (2 bullets per role)

## Replace resume PDF
Replace `public/resume.pdf` with your real resume PDF (keep the same filename).
